# orbit

Shared TypeScript package in the Polar frontend workspace. Provides cross-cutting utilities and types used by web applications and other packages.

## Key Concepts

- **Shared exports** -- Entry point at `src/index.ts` re-exports utilities consumed by `apps/web/` and other packages.

## Usage

Imported by other packages in the `clients/` workspace via the pnpm workspace protocol.
