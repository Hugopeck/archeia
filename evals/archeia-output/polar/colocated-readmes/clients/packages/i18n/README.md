# i18n

Internationalization package for the Polar frontend. Provides translation utilities, locale management, and localized string formatting for the web dashboard.

## Key Concepts

- **Translation files** -- Locale-specific translation strings for UI elements.
- **Vitest tests** -- `vitest.config.ts` configures unit tests for i18n utilities.

## Usage

Imported by `clients/apps/web/` for localized UI strings and number/date formatting.
